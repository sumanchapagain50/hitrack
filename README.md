# HeatTrack Nepal Dashboard

A real-time heat index evaluation platform for the Nepal Terai region.

## Features
- **Real-time Data**: Fetches live temperature and humidity from the Open-Meteo API.
- **Dual-Model Evaluation**:
  - **Humidex**: The Canadian standard for perceived heat.
  - **Steadman Model**: The US National Weather Service standard Heat Index (HI).
- **Interactive Visualization**:
  - **Geospatial Map**: Color-coded markers based on heat danger.
  - **Analytics**: Bar chart comparing different index results across the Terai belt.
  - **Regional List**: Detailed linear breakdown for major cities.

## Technologies Used
- HTML5 / Vanilla CSS
- JavaScript (ES6+)
- Leaflet.js (Mapping)
- Chart.js (Data Visualization)
- Open-Meteo API (Weather Data)

## How to Run
Simply open `index.html` in any modern web browser.
